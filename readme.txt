== Description ==

Add custom css in your site. Follow given simple steps to get your task done.
	
1. Install Advanced Custom CSS.
2. Activate Plugin.
3. Insert CSS code in respected code blocks.
4. Save settings
5. And you done!
	

**Control when to add CSS in page source**


*Set Global CSS*
You can set global CSS rules that will be added to every page of website.

*Set Post specific CSS*
You can set seperate CSS rules that will be added only when single post is displayed.

*Set Page specific CSS*
Similar to Post CSS, You can set seperate CSS rules that will be added only when single page is displayed.

== Frequently Asked Questions ==

1. Added CSS rules makes no effect.

== Installation ==

Use built in automatic installer or,

1. Copy the plugin folder into /wp-content/plugins
2. Activate plugin via the plugins page in the WordPress admin
3. Go to Settings to configure.
